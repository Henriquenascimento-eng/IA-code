# 🤖 IA Coder

Um assistente de inteligência artificial desenvolvido para auxiliar estudantes e desenvolvedores iniciantes no aprendizado de programação.

O projeto utiliza **Streamlit** para criar uma interface web interativa e integra a API da **Groq** para comunicação com modelos de linguagem (LLM), permitindo que o usuário tire dúvidas sobre programação em tempo real.

##  Funcionalidades

- 💬 Chat interativo com IA
- 🐍 Suporte para dúvidas de Python
- ☕ Suporte para dúvidas de Java
- 📚 Explicações passo a passo de códigos
- 🧠 Auxílio com algoritmos, estruturas de dados e bibliotecas
- 💡 Sugestões de boas práticas de programação

##  Tecnologias utilizadas

- Python
- Streamlit
- Groq API
- LLM (Large Language Model)

##  Como executar o projeto
  copie e cole no google: http://localhost:8501/
  ou clone o repositório e rode no terminal " py -m streamlit run main.py"
